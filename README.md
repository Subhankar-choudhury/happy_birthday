# 🎂 Happy Birthday Sarayu — Interactive Birthday Website

A small interactive birthday website created to celebrate **Sarayu's birthday**.
The page includes animations, surprises, music, and interactive elements to make the experience feel special and personal.

---

# ✨ Features

### 🥤 Coke Bottle Loading Screen

When the website first loads, a **Coke-themed loading screen** appears with a bouncing bottle animation and the message:

> *"Pouring some happiness..."*

---

### ⌨️ Typewriter Birthday Title

After the loader disappears, the title **"Happy Birthday Sarayu 🎂"** appears with a **typewriter animation**, giving a cinematic reveal effect.

---

### 🫧 Floating Background Bubbles

Soft **floating bubbles** continuously move in the background to create a dreamy aesthetic atmosphere.

---

### 🎉 Celebrate Button

Clicking the **Celebrate 🎉** button will:

* Display a **birthday cake image**
* Trigger **confetti animation**
* Start playing **background birthday music**

---

### 🎁 Further More Surprises

Clicking **Further more surprises…** reveals a sequence of surprises:

1. **Image 1 appears** wishing Sarayu a Happy Birthday
2. After a few seconds, the final surprise appears:

   * **Snuggles & Toffy image**
   * Final birthday message
3. Another **confetti celebration**

---

# 🚀 How to Run

1. Download or clone the project.
2. Ensure all files are in the same folder.
3. Double-click:

```

```

The website will open in your browser.
